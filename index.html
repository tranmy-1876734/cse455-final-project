Problem

With a sudden lockdown, many people were forced to work and do online schooling. Scanning documents is one of the additional tasks that many people encounter by working from home, and not all of them have a scanner at home. Although there are many scanner applications in the Google stores, not many of them provide a great quality of e-documents. Thus, we would like to attempt to provide a more reliable document scanner that is much accessible to other people.

Solution

Our current idea is to use the edge detector algorithm to separate the documents from the environment, use a filter to reduce some noise, then sharpen the entire image to give a better resolution. We use OpenCV as our main API for this project.


Techniques

We apply the image processing techniques learned in class to build a scanning app with high quality:

  • Change image to grayscale and blur image with openCV using Gaussian Blur
  • Use Canny Edge detection to detect edges to find paper
  • Use Contour to find the largest area in the image to find paper
  • Capture the image then process it
  • Save the scanned image
  • Change saved image to PDF

Tools

OpenCV, NumPy, Python Imaging Library (PIL)

Discussion

What problems did you encounter?

Webcam quality was low. It is challenging for the document output to be clear when the input image did not clearly show the texts from the document.

Are there next steps you would take if you kept working on the project?

We would add the option to convert the output image into a pdf document if desired.

How does your approach differ from others? Was that beneficial?

We managed to get higher quality of scan by experimenting different image processing techniques.
